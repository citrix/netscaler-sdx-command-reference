# ntpsync

NTP Sync

[show](#show%20ntpsync) | [set](#set%20ntpsync)

## show ntpsync

Use this operation to get status of ntpd

## Synopsys 

show ntpsync

## set ntpsync

Use this operation to enable/disable ntpd

## Synopsys {#synopsys-1 style="margin-left: 20px;"}

set ntpsync \[ntpd\_status=(false | true)\]

## Parameters 

**ntpd\_status**

ntpd status
